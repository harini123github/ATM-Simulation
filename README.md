# ATMSimulation - Java Console App

A simple Java-based ATM simulation that allows users to check balance, withdraw or deposit money, change PIN, and view transaction history â€” all from a terminal interface.

## Features

- PIN Authentication with 3 attempt limit
- Balance Inquiry
- Cash Withdrawal
- Cash Deposit
- Change PIN
- View Transaction History
- Persistent storage using a local text file (`atm_state.txt`)
---

## Tech Stack

- Language: **Java**
- I/O: **FileReader, FileWriter, BufferedReader, BufferedWriter**
- Data Storage: **Plain Text File**
---

## Getting Started

### Compile the code:
javac ATMSimulation.java

### Run the code:
java ATMSimulation
